Chapitre 1. Conditions préalables

1.1. Portée

1.1.1. Général

Quba-Viewer est un logiciel open source pour Mac OS, Windows et Linux, disponible en anglais, français et allemand, permettant d'afficher hors ligne des factures électroniques structurées et hybrides. « Structuré » désigne le format XML (UBL 2.1, CII 1.0/2016b) et « hybride » désigne du XML intégré dans un PDF (Factur-X/ZUGFeRD). Nous couvrons au minimum la partie EN16931 (XRechnung) et prenons en charge Factur-X 1/ZUGFeRD 2.x ainsi qu'Order-X.

1.1.2. Mentions légales

Aucun conseil fiscal ou juridique ne peut être donné ou déduit de l'utilisation de ce logiciel. La portée couvre la plupart des factures électroniques à traiter en Allemagne (depuis le 01.01.2025 conformément à la Wachstumschancengesetz, p. 23f, Art. 23).

1.1.3. Factures hybrides

Les auteurs de factures électroniques peuvent utiliser ce logiciel pour vérifier que les informations du XML correspondent à celles du PDF. Lors des ZUGFeRD-Entwicklertage 2025, un incident a été mentionné où quelqu'un a vérifié le PDF visuel d'une facture hybride, puis a payé le XML frauduleux et perdu 60 000€.

1.1.4. Archivage

À la réception de votre première facture électronique, vous pouvez être soumis à des obligations d'archivage numérique comme dans les GoBD allemandes. Malheureusement, Quba ne vous aide pas encore à respecter ces obligations.

1.2. Support

Support communautaire via Github et le Google Group. Support commercial via usegroup via Mustangproject. Pour anonymiser vos factures, utilisez le script Python d'Andreas Starke.

1.3. Systèmes d'exploitation supportés

Sous

  • Windows, versions x64 : 10, 11, Server 2022, Server 2025

  • Linux x64 : Debian 12, Ubuntu 24.04.2, Ubuntu 24.10

  • Mac OS : 14.x (Sonoma), 15.x (Sequoia)

Merci de nous signaler si le logiciel fonctionne sur d'autres plateformes.

1.4. Licence

Logiciel open source gratuit, y compris commercial, sous Apache Public License 2. Aucune garantie n'est fournie.

1.5. Protection des données

Quba-Viewer est une application hors ligne. Aucune donnée d'utilisation n'est collectée. En cas de connexion Internet, Quba peut vérifier les nouvelles versions au démarrage via votre adresse IP sur un serveur Github.

La validation nécessite la transmission de la facture à un serveur Internet. Sans abonnement Mustangserver, soumettez uniquement des factures de test ou anonymisées (einvoice-anonymizer). Il n'existe pas de version « Pro » ou commerciale de Quba-Viewer.

Sans contrat commercial, aucune mesure technique/organisationnelle ni accord de niveau de service n'est en place. Voici à quoi ils ressemblent pour Mustangserver.

1.6. Alternatives

La Open XRechnung Toolbox est basée sur la xrechnung-visualization mais sans support Factur-X Extended. Notre projet sœur Mustang propose la conversion PDF et la validation hors ligne. Le ZUGFeRD-Manager offre une interface graphique pour Factur-X. Elster propose une visualisation en ligne.

Chapitre 2. Téléchargement et lancement

2.1. Droits administrateur

Vous pouvez utiliser la version Windows sans droits administrateur en extrayant l'exe (quba-setup…​exe) avec 7-zip, puis en extrayant app-64.7z depuis $PLUGINSDIR et en lançant quba.exe.

2.2. Arguments de ligne de commande

Vous pouvez passer un nom de fichier en argument pour installer Quba comme visionneur par défaut pour *.xml et/ou *.pdf.

2.3. Ouvrir des fichiers

Trois façons d'ouvrir un fichier de facture :

  • Menu : Fichier > Ouvrir

  • Raccourci clavier : Ctrl+O (Windows/Linux) ou Cmd+O (Mac OS)

  • Glisser-déposer : déposer un fichier XML ou PDF dans la fenêtre Quba

Quba prend en charge les fichiers XML (XRechnung, UBL, CII, Factur-X) et les PDF hybrides avec XML intégré.

2.4. Onglets

Quba permet d'ouvrir plusieurs factures simultanément dans des onglets.

  • Nouvel onglet : bouton + dans la barre d'onglets, puis sélectionner un fichier. En cas d'annulation, l'onglet vide est automatiquement fermé.

  • Fermer un onglet : icône ×. La fermeture du dernier onglet revient à l'écran d'accueil.

  • Changer d'onglet : cliquer sur l'onglet souhaité.

2.5. Changer de langue

Quba est disponible en anglais, français et allemand. Menu Langue pour changer. L'affichage se met à jour immédiatement sans redémarrage.

2.6. Imprimer XML et PDF

Imprimer|XML visualise la partie XML, Imprimer|PDF imprime le PDF lisible d'une facture hybride. Imprimer|PDF est désactivé sans PDF disponible (p.ex. XRechnung).

2.7. Afficher les Business Terms (BT)

Via Fichier|Afficher les IDs, affichez les identifiants BT à côté des champs de texte.

2.8. Recherche

Appuyez sur Ctrl+F pour ouvrir la barre de recherche.

  • Navigation : boutons > (suivant) et < (précédent).

  • Recherche dans la visualisation HTML et, si ouvert, dans le PDF.

  • En l'absence de résultat, un message explicite s'affiche.

  • Fermer avec Échap.

2.9. Pièces jointes

Les PDF hybrides peuvent contenir des pièces jointes intégrées. Quba les détecte et les affiche dans la vue de la facture. Un clic ouvre la pièce jointe dans l'application appropriée. En cas d'erreur, un message s'affiche.

2.10. Notifications de mises à jour

Quba vérifie les nouvelles versions au démarrage (sauf si désactivé, voir Désactiver les mises à jour automatiques).

  • Mise à jour disponible : bannière d'information en haut de la fenêtre.

  • Mise à jour téléchargée : cliquer sur Redémarrer pour l'appliquer, ou redémarrer manuellement plus tard.

Chapitre 3. Fonctionnalités entreprise

3.1. Installation sur serveurs terminaux/pour tous les utilisateurs

Des versions MSI sont disponibles sur Github. Installation pour tous les utilisateurs :

MsiExec /i "Quba_1.5.1.msi" AUTOUPDATE=false MSIINSTALLPERUSER="" ALLUSERS=2 /L*v Install.log

3.2. Installation headless

Après une installation MSI, Quba démarre par défaut. Pour une installation silencieuse, modifier le MSI avec instedit et supprimer l'action runAfterFinish.

3.3. Désactiver les mises à jour automatiques

Dans \AppData\Local\Programs\Quba\AppConfig.ini, définir performOnStartup=false :

[AutoUpdate]
performOnStartup=true

Utile si vous distribuez Quba comme package logiciel certifié par votre équipe.

Glossaire

BT

Business Term (terme métier)

CII

Cross Industry Invoice

Factur-X

Format de facture électronique hybride franco-allemand intégrant du XML CII dans un PDF (identique à ZUGFeRD 2.x)

MSI

Microsoft Installer

UBL

Universal Business Language

XRechnung

Format national allemand de facture électronique basé sur EN16931, utilisant UBL ou CII

ZUGFeRD

Format de facture électronique hybride allemand intégrant du XML CII dans un PDF (identique à Factur-X)
